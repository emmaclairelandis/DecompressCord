# DecompressCord

Discord users on Android who upload .jpg files to their friends using the app will experience their files being compressed and ruined. Besides the alternatives of using third-party hosting and only uploading files from desktop, DecompressCord seeks to amend this issue.

DecompressCord will convert all .jpg files to the highest possible quality .png file under Discord's 10 MB limit, lowering the resolution as little as possible to retain image quality.
